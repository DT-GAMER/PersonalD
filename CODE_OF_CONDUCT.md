# Code Of Conduct

PersonalD should be a welcoming project for people learning, building, and contributing in good faith.

## Expected Behavior

- Be respectful and patient
- Give constructive feedback
- Assume good intent when possible
- Keep discussions focused on the project
- Welcome beginners and explain context when it helps

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks
- Discrimination or exclusionary language
- Sexualized language or unwanted attention
- Publishing private information without permission
- Spam, trolling, or intentionally disruptive behavior

## Enforcement

Maintainers may remove comments, close issues or PRs, block users, or take other reasonable action to protect the project community.

If you need to report a problem, contact a maintainer privately where possible. If private contact is not available yet, open a GitHub issue with only the minimum safe context and ask for maintainer contact.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and community channels related to PersonalD.
